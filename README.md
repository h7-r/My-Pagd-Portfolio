## 🌱 기획 의도
> 디자인보다 **내용과 흐름이 자연스럽게 전달되는 UI**를 목표로 한 포트폴리오입니다.  
> 사용자가 페이지를 읽어 내려가며  
> **생각의 흐름이 끊기지 않도록** 구조 설계에 집중했습니다.

---

## 🌿 디자인 방향
초기에는 차분하면서도 인상을 남기는 **보라색**을 메인 컬러로 설정했습니다.  
하지만 전체 페이지를 바라보았을 때  
시각적인 피로도가 높고, 내용에 집중하기 어렵다고 판단했습니다.

> 그 결과,  
> **밝지 않으면서도 안정적인 녹색 계열**을 메인 컬러로 선택하여  
> 페이지를 읽는 흐름이 더 편안하게 이어지도록 구성했습니다.

---

## 🧩 UI · 구조
- **Flex / Grid 기반 레이아웃**
- PC · Tablet · Mobile 환경을 고려한 반응형 설계
- 콘텐츠 흐름을 우선한 구조 배치
- 유지보수를 고려한 반복 구조 정리

> 구조는 디자인을 드러내기 위한 수단이 아니라,  
> **내용을 방해하지 않기 위한 틀**이라고 생각했습니다.
