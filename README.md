# My-Pagd-Portfoliogit

### 기획 의도
디자인보다는 **내용과 흐름이 자연스럽게 전달되는 구조**에 집중했습니다.  
화려한 효과보다는 사용자가 읽고 이해하기 쉬운  
레이아웃과 정보 배치를 우선으로 설계했습니다.

### 디자인 방향
메인 컬러로는  
차분하면서도 기억에 남는 인상을 주는 **보라색**을 사용하여 진행하였으나 보라색의 밝음이
제가 원했던 내용과 흐름을 전달하기에 어려웠고 페이지를 봤을때 피로가 덜한 녹색을 선택하였습니다.

### UI · 구조
레이아웃은 **Flex**와 **Grid**를 기반으로 구성했으며,  
PC · 태블릿 · 모바일 환경에서  
읽기 편한 흐름이 유지되도록 **반응형을 고려**했습니다.
최대한 그리드와 flex을 활용하였고 설계와 유지보수의 신경을 쓰려고 노력하였습니다. 

### 부족한점과 깨닳은 점
아직은 배운는 단계이며, css와 반응형에서 문제가 많았고 구현전 마크업당시 전체적인 부분을 체크하지 못한 부분이 있었습니다.
각각의 넓이와 높이의 중요성도 알게되었고 그리드 활용시 한번더 생각해야한다는것과 clamp 속성에 대해 다시한번더 공부할수있었습니다.

### 목표
부족한 이론공부가 필요하기에 꾸준히 공부하고 나아가는 작은 목표를 기준으로
다음 구현시 더나은 방안과 더 빠른 구현으로 신속하고 정확하게 구현하고 노력할 것 이며, 언젠가
저의 UI를 사용하게 됬을때 이용자분들께서 여운이 남는 그런 UI 만드는 것이 큰 목표입니다.
