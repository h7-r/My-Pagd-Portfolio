# My-Pagd-Portfoliogit
<h1>첫 웹 페이지를 만들어보다 </h2>

### 기획 의도
디자인보다는 **내용과 흐름이 자연스럽게 전달되는 구조**에 집중했습니다.  
화려한 효과보다는 사용자가 읽고 이해하기 쉬운  
레이아웃과 정보 배치를 우선으로 설계했습니다.

### 디자인 방향
메인 컬러로는  
차분하면서도 기억에 남는 인상을 주는 **보라색**을 사용해  
전체적으로 개인적인 분위기를 유지했습니다.

### UI · 구조
레이아웃은 **Flex**와 **Grid**를 기반으로 구성했으며,  
PC · 태블릿 · 모바일 환경에서  
읽기 편한 흐름이 유지되도록 **반응형을 고려**했습니다.

### 목표
이 포트폴리오를 통해  
단순한 구현이 아니라  
**왜 이렇게 설계했는지, 사용자가 어떻게 느낄지를 고민하는 과정**을  
보여주고자 했습니다.
